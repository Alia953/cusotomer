Customer Support System

Instructions:
1. Clone or download the repository.
2. Navigate to the folder containing the project.
3. Run the following command to start the system:
   python main.py
4. Choose the desired option from the menu:
   1. Create a new support ticket.
   2. View all support tickets.
   3. Mark a ticket as resolved.
   4. Exit the system.

Features:
- You can create a support ticket with a name and issue description.
- View all tickets with their status.
- Resolve tickets by marking them as resolved.
